Detailed Description

Management namespace of pin mapping.

Please give to the standard Arduino libraries methods the values defined in this namespace. The methods are evaluated at compile time, so there is no overhead at runtime.

Note
It helps your understanding to refer the PLEN2's circuit and schematic. -> https://github.com/plenprojectcompany/plen-ControlBoard
See also
Arduino Micro's pin mapping -> http://arduino.cc/en/Hacking/PinMapping32u4
